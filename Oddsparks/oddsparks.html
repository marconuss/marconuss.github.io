About

This time at Massive Miniteam I had the opportunity to work on Oddsparks, an in-house developed game published by Handy Games.
Oddsparks combines automation and RTS elements in a unique way, making the automation genre approachable and fun for all players.
My main tasks were to port the Demo of the game to PlayStation 5 and Xbox Series X/S, with the addition of new features, like the haptic feedback system for the DualSense controller.
I also had the opportunity to be part of the development team directly, and I was able to support with bug fixing and the implementation of editor tools for the artists.

The game was developed using Unreal Engine 5 and C++
